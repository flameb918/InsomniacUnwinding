## 🖥️ What this is

InsomniacUnwinding is a Windows sleepmask based on Ekko. It keeps unwind data at sleep time, which helps the app restore state with less work after sleep or pause cycles.

This project is for users who want a simple way to run a sleepmask build on Windows without setting up a full development environment.
